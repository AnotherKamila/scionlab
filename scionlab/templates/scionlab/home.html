SCIONLab is a global research network to test the SCION next-generation internet architecture. The main idea of SCIONLab is that participants join the SCION network with their own computation resources and can easily set up and run their own full-fledged SCION autonomous systems (ASes).

These ASes will actively participate in routing in the SCIONLab network, SCIONLab enables realistic experimentation with the unique properties of SCION architecture.
